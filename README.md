# Интеллектуальная система подбора параметров темперирования шоколада

## Описание проекта
ML-продукт для кондитерского производства, прогнозирующий 6 ключевых параметров техпроцесса (температуры и время этапов) на основе состава сырья. Цель — снижение процента брака и повышение стабильности качества.

## Датасет
- Самостоятельно собран из научных источников (Scopus, Journal of Food Engineering)
- Признаки: содержание какао-масла, молочного жира, сахара, лецитина, сухих веществ какао, тип использования
- Целевые переменные: температуры и время трёх этапов производства (6 выходов)

## Модель
- Сравнение линейной регрессии, Random Forest и полносвязной нейросети (MLP)
- Оптимальная архитектура: 3 слоя (128 → 64 → 32), активация ReLU, Dropout для регуляризации
- Фреймворк: PyTorch

## Результаты
- R² > 0.95 для температурных параметров
- R² ~ 0.85 для временных параметров
- Валидация: MSE, MAE, кривые обучения, кросс-валидация

## Интеграция
Модель упакована в Telegram-бота. Оператор вводит состав сырья и получает рекомендации в реальном времени.
